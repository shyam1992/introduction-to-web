Who is this series for / Prerequisites

This series is to teach you how to develop web apps. You do not need to know anything about frontend languages previous to this series.

You should have some familiarity with computers and the Internet.

This series will teach you HTML, CSS, and JavaScript. All three are necessary components to start making modern web apps. If you need help in any of these categories, this series and the references should be helpful. If you already know how to code (like Java or python) and want to veer more into frontend development, this series will also be helpful.

If you think, you're well aware of the basics of web development then you can skip this.
