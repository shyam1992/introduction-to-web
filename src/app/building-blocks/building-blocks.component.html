BUILDING BLOCKS

There are three key blocks involved in building a web page, they are HTML, CSS and Javascript.

Ideally, nothing else runs on the browser and nothing else you need to know to build web sites or web applications.

Website is a simple static page with no interaction, whereas, Web application is a complete package of information and human interaction like form submission.

Let us think of building an electric car, which will need a body, color and shape, and the motor with connection to wheels. If we go with this analogy, HTML becomes the base building body, CSS is responsible for shape and colors of the car, and Javascript is the core of functional aspects
